## Search Engine Project
### В данном репозитории содержится код реализации поискового сервера на языке С++.
---
### Краткое описание работы программы:

Программа получает поисковые запросы из файла requests.json .
По этим запросам осуществляется поиск в текстовых файлах, расположенных в папке resources.
Максимальное количество возможных документов в ответе задается в файле config.json .
Результаты поиска ранжируются, сортируются и записываются в файл answers.json .

### Установка и сборка программы:

1. Клонируйте репозиторий:

	```bash
	git clone https://github.com/base826/search_engine.git
	cd search_engine
	```

2. Соберите проект:

Сборка осуществляется в терминале, из папки проекта.
Перед сборкой проекта необходимо проверить правильность путей к конфигурационным и исходным текстовым файлам в файлах config/config.json и
include/converter.h .
Сборка осуществляется следующими командами:

```
	cmake -S . -B build
	cmake --build build
```
После сборки программы вы сможете запустить её исполняемым файлом search_server.

### Структура проекта:

- `congig/congig.json`: конфигурационный файл, в которм указаны пути к текстовым файлам в которых будет производиться поиск, и количество документов в ответе.
- `config/requests.json`: файл с поисковыми запросами
- `config/answers.json`: файл, в который записываются результаты поиска
- `resources/`: папка с текстовыми файлами, в которых будет производиться поиск
- `tests/`: папка с google-тестами, для проверки правильной работоспособности программы
- `include/`: папка с заголовочными файлами .h
- `src/`: папка с файлами .cpp
