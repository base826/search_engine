## Search Engine Project
### В данном репозитории содержится код реализации поискового сервера на языке С++.
---
### Краткое описание работы программы:

Программа получает поисковые запросы из файла requests.json .
По этим запросам осуществляется поиск в текстовых файлах, расположенных в папке resources, 
ссылки на которые указаны в файле config.json .
Максимальное количество возможных документов в ответе также задается в файле config.json .
Результаты поиска ранжируются, сортируются и записываются в файл answers.json .
Данные файлы располагаются в папке config .

### Сборка программы:

Сборка программы осуществляется в терминале, из папки проекта.
Перед сборкой проекта необходимо проверить правильность путей к конфигурационным и исходным текстовым файлам в файлах config/config.json и
include/converter.h .
Сборка осуществляется следующими командами:
'''
	cmake -S . -B build
	cmake --build build
'''
После сборки программы вы сможете запустить её исполняемым файлом search_server.exe
